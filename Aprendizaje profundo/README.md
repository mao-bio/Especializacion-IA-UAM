# Aprendizaje Profundo (Deep Learning)

Estudio de redes neuronales profundas, arquitecturas avanzadas y sus aplicaciones en visión por computador y procesamiento de secuencias.

## Unidades Temáticas

*   **Unidad 1: Introducción**: Conceptos básicos y perceptrón multimapa.
*   **Unidad 2: Redes Neuronales**: Implementación y optimización de redes densas.
*   **Unidad 3: CNNs**: Redes Neuronales Convolucionales para clasificación de imágenes.
*   **Unidad 4: GANs**: Redes Generativas Adversarias.
*   **Unidad 5: Modelos Secuenciales**: RNNs y LSTMs para datos temporales.
*   **Unidad 6: Transformers**: Mecanismos de atención y arquitecturas Transformer.

## Proyectos Destacados
*   Clasificación de pozos petroleros (Deep Learning aplicado).
*   Generación de imágenes con GANs.
*   Análisis de series temporales.

## Tecnologías
*   TensorFlow / Keras
*   PyTorch
*   Python
