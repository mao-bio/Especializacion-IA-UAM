# Visualización para Grandes Volúmenes de Datos

Técnicas y herramientas para la representación visual de datos complejos y masivos (Big Data).

## Contenido

*   **Unidad 1: Introducción**: Conceptos de Big Data y su relación con la visualización.
*   **Unidad 2: Modelado**: Procesamiento y gestión de datos a gran escala.
*   **Unidad 3: Técnicas de Visualización**: Uso de librerías avanzadas para contar historias con datos (Storytelling).
*   **Unidad 4: Proyecto de Aplicación**: Visualización de datos reales (Ej: Datos geográficos del Departamento de Caldas).

## Herramientas
*   Power BI / Tableau
*   Matplotlib / Seaborn / Plotly (Python)
*   ggplot2 (R)
*   Herramientas de gestión de Big Data.
