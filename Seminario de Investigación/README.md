# Seminario de Investigación

Este módulo comprende el desarrollo y formulación de un proyecto de investigación aplicado en el campo de la Inteligencia Artificial.

## Fases del Proyecto

El seminario se divide en tres etapas clave:

1.  **Problematización**: Identificación y planteamiento del problema de investigación.
2.  **Aplicación**: Formulación del anteproyecto y metodología.
3.  **Finalización**: Presentación de resultados y sustentación del proyecto.

## Proyecto Destacado
**Sistema de IA para predicción y monitoreo de EDA**

Este proyecto busca aplicar modelos de inteligencia artificial para la predicción temprana y el monitoreo de casos, optimizando la respuesta en salud pública.
